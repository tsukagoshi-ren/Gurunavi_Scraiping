# ぐるなび店舗情報スクレイピングツール Professional v2.0

## 概要
日本最大級のグルメサイト「ぐるなび」から店舗情報を効率的に収集するデスクトップアプリケーションです。

## 🆕 v2.0 の新機能
- **Seleniumベース**: JavaScript対応の高精度スクレイピング
- **タブ式UI**: 検索・設定・ログの整理されたインターフェース
- **詳細設定**: ブラウザ設定、アクセス制御の細かな調整
- **ログ機能**: リアルタイムログ表示と保存機能
- **統計レポート**: Excel出力時の詳細分析シート
- **自動ChromeDriver**: webdriver-managerによる自動管理

## 機能
- エリア・ジャンル・キーワードによる柔軟な検索
- 店舗名、電話番号、住所、ジャンル、営業時間等15項目の詳細情報取得
- Excel形式での結果出力（複数シート対応）
- GUI による直感的な操作
- 取得件数制限機能（1-1000件）
- リアルタイムプレビュー
- 設定カスタマイズ（JSON設定ファイル）

## 使用方法

### 1. 実行ファイル版（推奨）
```bash
# 1. 配布パッケージを展開
# 2. final_build.bat を実行（初回ビルド時）
# 3. GurunaviScraperPro.exe を実行
```

### 2. Python環境での実行
```bash
# 必要なパッケージのインストール
pip install -r requirements_full.txt

# ChromeDriverセットアップ
python chromedriver_setup.py

# アプリケーション実行
python production_ready_scraper.py
```

### 3. 開発者向けビルド
```bash
# 完全ビルド（推奨）
final_build.bat

# カスタムデプロイ
python deploy.py
```

## 取得可能な情報
- 店舗名、電話番号、住所、ジャンル

## トラブルシューティング

### よくある問題と解決方法

1. **アプリが起動しない**
   - Windows Defenderの除外設定を確認
   - 「詳細情報」→「実行」で許可
   - 管理者権限で実行

2. **Seleniumエラー**
   - Chromeブラウザを最新版に更新
   - 「ChromeDriver更新」ボタンを実行
   - ファイアウォール設定を確認

3. **データが取得できない**
   - インターネット接続を確認
   - 検索条件を変更して再試行
   - ログタブでエラー詳細を確認

4. **保存エラー**
   - 保存先フォルダの書き込み権限を確認
   - ファイル名の使用禁止文字をチェック
   - 既存ファイルが開かれている場合は閉じる

■ 概要
このツールは、日本最大級のグルメサイト「ぐるなび」から
店舗情報を効率的に収集するための専用アプリケーションです。

■ 動作環境
- OS: Windows 10/11 (64bit)
- RAM: 4GB以上推奨
- HDD: 1GB以上の空き容量
- インターネット接続: 必須
- Google Chrome: 最新版推奨

■ クイックスタート
1. 配布パッケージを適当なフォルダに展開
2. GurunaviScraperPro.exe をダブルクリック
   （初回起動時はChromeDriverが自動インストールされます）
3. 検索条件を設定して「スクレイピング開始」をクリック

■ 詳細操作手順

【検索条件設定】
1. 「検索・実行」タブを選択
2. 検索条件を設定:
   - 都道府県: 必須（プルダウンから選択）
   - 市区町村: 任意（詳細地域指定）
   - ジャンル: 任意（料理カテゴリ）
   - 最寄り駅: 任意（駅名）
   - キーワード: 任意（自由検索語）
   - 最大件数: 1-1000件（推奨: 100件以下）

【保存設定】
- 保存先: デフォルトはダウンロードフォルダ
- ファイル名: 自動で日時が付与されます

【実行】
1. 「スクレイピング開始」ボタンをクリック
2. 進行状況をリアルタイムで確認
3. 完了後、指定場所にExcelファイルが保存されます

【詳細設定（上級者向け）】
「詳細設定」タブで以下を調整可能:
- ヘッドレスモード: ON推奨（処理高速化）
- アクセス間隔: 2-5秒（サーバー負荷軽減）
- タイムアウト設定: 15秒推奨
- ユーザーエージェント: カスタム設定可能

【ログ確認】
「ログ」タブで処理状況とエラー詳細を確認できます

■ 取得される情報
- 基本情報: 店舗名、電話番号、住所、ジャンル
- アクセス: 最寄り駅、駐車場情報
- 営業情報: 営業時間、定休日、座席数
- サービス: 個室、禁煙・喫煙、クレジットカード対応
- その他: 予算、店舗URL、取得日時

■ Excel出力シート構成
1. 店舗データ: 全取得データの一覧
2. 統計情報: データ品質と取得状況の統計
3. ジャンル別集計: 料理カテゴリごとの集計
4. エリア別集計: 都道府県別の集計

■ トラブルシューティング

【起動しない場合】
- Windows Defenderの除外設定を確認
- 管理者権限で実行
- Chromeブラウザを最新版に更新

【取得できない場合】
- インターネット接続を確認

- 「ChromeDriver更新」ボタンを実行

【動作が重い場合】
- 最大取得件数を50件以下に設定

【エラーが発生した場合】
- 「ログ」タブでエラー内容を確認
- アプリを再起動
- 設定をデフォルトに戻す
